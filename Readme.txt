This project, Bookify, is a front-end web application designed for managing and displaying book collections. Built using HTML, CSS, and JavaScript, it allows users to view book details, add new books, and delete existing ones.

Key Features:
Add Book: Users can input a book's title, author, and description through a popup form.
Delete Book: Books can be removed from the list with a simple delete button.
Responsive Design: Clean and responsive layout for better user experience.
Interactive UI: Dynamic popups and smooth interactions for managing books.
This project focuses on user-friendly book management using only front-end technologies.
